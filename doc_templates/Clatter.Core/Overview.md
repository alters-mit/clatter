# Overview

`Clatter.Core` contains most of the logic for Clatter. For a specialized Unity extension library, see `Clatter.Unity`.

To get started with `Clatter.Core`, read the following documentation:

1. `AudioObjectData` Per-object audio data.
2. `CollisionEvent` A collision between two objects.
3. `AudioEvent` Abstract base class for audio events.
4. `Impact` An impact audio event.
5. `Scrape` A scrape audio event.
6. `Creator` Entry point for generating audio.
7. `AudioGenerator` Multi-threaded audio generator.
